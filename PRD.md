# 프로젝트: 간단한 웹 퍼즐 게임

## 목표
- 브라우저에서 실행되는 간단한 퍼즐 게임을 만든다.
- 초보자도 코드를 읽고 수정하기 쉽게 구성한다.

## 기능
- 게임 보드를 화면에 표시한다.
- 사용자의 클릭 이벤트를 처리한다.
- 같은 카드 짝을 맞추는 4x4 퍼즐 게임으로 만든다.
- 게임 진행 상태를 화면에 보여준다.
- 다시 시작 버튼으로 게임을 초기화할 수 있다.

## 기술
- HTML
- CSS
- JavaScript

## 대상
- 초보자도 이해할 수 있는 수준으로 작성한다.

## 폴더 구조

```text
puzzle-game/
├─ index.html
├─ css/
│  └─ style.css
├─ js/
|  ├─ app.js   -- 기존 script.js에서 앱 시작점 관련 소스만 위치
|  ├─ board.js -- 퍼즐 보드 그리기 영역
|  ├─ game.js  -- 클릭이벤트, 정답확인, 클리어조건
│  └─ utils.js -- 랜덤섞기, 공통함수
├─ assets/
│  └─ images/
└─ PRD.md
```
